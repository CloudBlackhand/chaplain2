WhatsApp API - Documentação

Esta é a API de integração para o sistema de WhatsApp. Utilize os endpoints abaixo para interagir com o sistema.

Endpoints Disponíveis

GET /status

Verifica o status atual da conexão com o WhatsApp.

{
  "ready": true,
  "qrCode": null
}

GET /health

Verifica a saúde geral do sistema e suas estatísticas.

GET /clients

Lista todos os clientes disponíveis na base de dados.

GET /client/{sa}

Obtém detalhes de um cliente específico e seu histórico de mensagens.

POST /send-message

Envia uma mensagem individual para um cliente.

{
  "sa": "123456",
  "message": "Olá, tudo bem?"
}

POST /send-bulk

Envia mensagens em massa para múltiplos clientes.

{
  "sa_list": ["123456", "789012"],
  "message_template": "Olá {nome}, tudo bem?"
}

POST /webhook/setup

Configura o URL do webhook para receber notificações de mensagens.

Status do Sistema

Para monitorar o status do sistema, acesse /health.
